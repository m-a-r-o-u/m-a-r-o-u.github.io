# m-a-r-o-u.github.io

Personal landing page.

## Content editing

Section content is loaded from Markdown files:

- `content/cv.md`
- `content/contact.md`
- `content/projects/index.md`
- `content/projects/*.md` for individual project pages

Use hash routes for direct links:

- `/#cv`
- `/#projects`
- `/#contact`
- `/#projects/<project-slug>`
